# shubham
PUBG Weapon Analysis using AI & Machine Learning
🎯 Project Overview

This project performs a comprehensive analysis of PUBG weapon statistics using Python, Data Science, and Machine Learning techniques. The dataset contains detailed weapon information such as damage, range, bullet speed, magazine capacity, fire rate, and kill statistics. The goal is to discover hidden patterns, compare weapon performance, visualize weapon characteristics, and build predictive models for weapon effectiveness. The dataset is sourced from a PUBG weapon statistics collection on Kaggle.

📂 Dataset

Source:
[PUBG Weapon Stats Dataset](https://www.kaggle.com/datasets/aadhavvignesh/pubg-weapon-stats) (Kaggle)

Features Included
Weapon Name
Weapon Type
Bullet Type
Damage
Magazine Capacity
Range
Bullet Speed
Rate of Fire
Shots to Kill (Chest)
Shots to Kill (Head)
Damage Per Second (DPS)
Fire Mode
Body Damage Levels (BDMG)
Head Damage Levels (HDMG)

The dataset contains detailed weapon attributes that support exploratory data analysis and weapon performance comparisons.
